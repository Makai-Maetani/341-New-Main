Visual Hierarchy

Visual Hierarchy refers to guiding a user using UX or UI design to view content in a created order. There is visual Hierarchy in the design of Soundclouds website. We see as a company they want you to see an ad to join their subscription service. Below that is a the search bar which is the most probable item you will be using. Then there is a lot of suggestions to help you stay on their page.

Soundcloud

White Space

White space refers to empty space that lets the viewer breathe and take in the more impotant elements of a piece. Github uses this really well. It uses negative space to subtract the unimportant elements of a site and allows the space to highlight the important tools that you would probably be looking for.

Github

Contrast

Contrast is using noticable differences in elmements to create an appealing visual scene. In Replit we can see it used with a beautiful pictured description of what the website is next to a bright easy to navigate sign in section
